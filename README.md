# Overview
Frog:RIO is a game built using the [Love2D](https://love2d.org/) framework, assets from [Pixel Adventure](https://pixelfrog-assets.itch.io/pixel-adventure-1). It was created for educational purposes.

# Customize the levels
Frog:RIO offers level design capabilities. see [Documentation](https://github.com/lenlenlL6/Frog-RIO/wiki) for more information.
